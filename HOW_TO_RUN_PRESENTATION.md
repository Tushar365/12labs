# How to Run the Presentation

## Quick Start

### Option 1: Double-Click (Easiest)
1. Navigate to your project folder: `d:\projects\12labs\new1`
2. Double-click `presentation.html`
3. It will open in your default web browser

### Option 2: Use the Batch File (Windows)
1. Double-click `run_presentation.bat`
2. The presentation will open automatically

### Option 3: Use PowerShell Script
1. Right-click `run_presentation.ps1`
2. Select "Run with PowerShell"
3. The presentation will open

### Option 4: Command Line
Open PowerShell or Command Prompt in the project folder and run:

**PowerShell:**
```powershell
Start-Process presentation.html
```

**Command Prompt:**
```cmd
start presentation.html
```

**Or specify a browser:**
```powershell
# Chrome
Start-Process chrome.exe "presentation.html"

# Edge
Start-Process msedge.exe "presentation.html"

# Firefox
Start-Process firefox.exe "presentation.html"
```

## Tips for Best Experience

1. **Use Full Screen**: Press F11 for a distraction-free presentation
2. **Use a Modern Browser**: Chrome, Edge, or Firefox work best
3. **Projector Mode**: If presenting on a projector, use full-screen mode
4. **Navigation**: Practice using arrow keys for smooth transitions

## Troubleshooting

### If the presentation doesn't open:
- Make sure you're in the correct folder: `d:\projects\12labs\new1`
- Try right-clicking `presentation.html` ‚Üí "Open with" ‚Üí Choose your browser
- Check that your browser supports modern HTML5/CSS3

### If slides don't display correctly:
- Try a different browser (Chrome, Edge, or Firefox)
- Make sure JavaScript is enabled in your browser
- Check browser console for any errors (F12)
